# EBU法

## Variable Ⅱ
1セットで2度ずつ比較させる．

記号|意味(例)
:---:|:---:
n|原画の種類(1, 2, 3)
br|bitrate(15M, 20M, 40M)
p|エンコードパラメータ(3, 5, 6)
 
  | セット1 | セット2 | … |
----|----|----|----|
グレー  | グレー | グレー | …
A_n  | A_1 | A_1 
グレー  | グレー | グレー
B\_n\_br\_p  | B\_1\_15M\_3 | B\_1\_20M\_3
グレー  | グレー | グレー
A_n  | A_1 | A_1
グレー  | グレー | グレー
B\_n\_br\_p  | B\_1\_15M\_3 | B\_1\_20M\_3

のような感じで3x3x3の27セット行う．  
Aは原画、Bは評価対象．

## Variable Ⅰ
1セットを1度ずつ比較させる．最初の5セットくらいはわざと違いが分かりやすいようなものを選んで，それは評価から省く．
